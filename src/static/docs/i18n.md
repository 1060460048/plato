# I18n

简单的国际化实现

## 文件

- `plugins/i18n`
- `static/i18n/*.json`

## 用法

在 index.js 引入 I18n 插件

``` js
import Vue from 'vue'
import I18n from 'plugins/i18n'

// 使用插件
Vue.use(I18n)
```

在 App.vue 中设置 i18n 资源

``` js
export default {
  name: 'App',
  i18n: {
    resources: {
      "message": {
        "plato": "好的开始是成功的一半！"
      },
      "routes": {
        "about": "关于",
        "login": "登录",
        "logout": "注销"
      },
      "abc": "a{b}c"
    }
  }
}
```

在组件中使用资源

``` vue
<template>
{{__('message.plato')}}
{{__('abc', { b: 'B' })}}
</template>
```
