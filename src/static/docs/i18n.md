## I18n

简单的国际化实现

### 文件

- `plugins/i18n`
- `static/i18n/*.json`

### 用法

#### 在 index.js 引入 I18n 插件

``` js
import Vue from 'vue'
import I18n from 'plugins/i18n'

// 使用插件
Vue.use(I18n, {
  // 设置数据源
  data () {
    // 从 store 中获取，实现同步
    return env(store.state).i18n
  }
})
```

#### 在 App.vue 中设置 i18n 资源

``` js
export default {
  name: 'App',
  // 可选
  i18n: {
    // 如果配置，则会覆盖全局配置，与 .vue 的 data 不同
    data () {}
  }
}
```

#### 在组件中使用资源

``` vue
<template>
{{__('message.plato')}}
{{__('abc', { b: 'B' })}}
</template>
```
